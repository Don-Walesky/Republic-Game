# Wave 06: Legislature

**How the player faces opposition.**

**Duration**: 6-8 weeks

**Status**: 🔴 Not Started

**Depends On**: Wave 4 (Government) + Wave 5 (Economy)

**Milestone Gate**: "You must work with parliament"

---

## Objective

Build parliament system. Player can't just decree things. They must convince parliament. Opposition exists. Coalition building required.

By the end of Wave 6:
- Parliament exists with members
- Bills can be proposed
- Parliament votes on bills
- Opposition fights proposals
- Coalitions needed for majorities
- Constitutional limits enforced
- Ready for Wave 7 (military)

---
