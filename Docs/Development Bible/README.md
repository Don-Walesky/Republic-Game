# Republic Development Bible – Volume 1: Master Production Roadmap

**From first line of code to playable alpha.**

This is the master reference for all Republic development.

---

## What Is This?

A complete, wave-by-wave breakdown of:
- What gets built
- Why it gets built
- When it gets built
- How it integrates with other systems
- Success criteria for each Wave

**Everything else** (GitHub Issues, AI prompts, code) traces back to this document.

---

## Structure

### Master Overview
- Vision & principles
- Timeline summary
- Dependency chain
- Release strategy

### Wave Breakdown (Waves 0-10)

Each Wave directory contains:
- **README.md** - Wave overview and systems
- **Deliverables.md** - What ships at end of Wave
- **Technical Specs** - Implementation details
- **Integration Points** - How it connects to other Waves
- **Success Criteria** - How we know it works
- **GitHub Epics** - Converted to Issues
