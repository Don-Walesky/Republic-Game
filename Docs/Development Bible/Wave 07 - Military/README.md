# Wave 07: Military

**The ultimate power and ultimate cost.**

**Duration**: 4-6 weeks

**Status**: 🔴 Not Started

**Depends On**: Wave 4 (Government) + Wave 6 (Legislature)

**Milestone Gate**: "You control military power"

---

## Objective

Build military system. Player can deploy forces, start wars, experience consequences.

By the end of Wave 7:
- Armed forces exist
- Player can deploy forces
- Wars can break out
- Military spending impacts budget
- Wars affect approval
- Victory/defeat have consequences
- Ready for Wave 8 (diplomacy)

---
