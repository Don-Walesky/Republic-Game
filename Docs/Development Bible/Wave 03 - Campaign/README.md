# Wave 03: Campaign

**How the player gains power.**

**Duration**: 4-6 weeks

**Status**: 🔴 Not Started

**Depends On**: Wave 1 (Office) + Wave 2 (World)

**Milestone Gate**: "You can run for office"

---

## Objective

Build the electoral system. The player enters the world and campaigns to become the leader.

By the end of Wave 3:
- Player can announce candidacy
- Campaign mechanics work
- Elections happen
- Player can win or lose
- Winner transitions to government (Wave 4)
- Ready for Wave 4 (governing)

---
