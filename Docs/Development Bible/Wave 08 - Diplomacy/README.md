# Wave 08: Diplomacy

**Power multiplied through international relations.**

**Duration**: 4-6 weeks

**Status**: 🔴 Not Started

**Depends On**: Wave 7 (Military) - diplomacy backed by force

**Milestone Gate**: "You can shape international politics"

---

## Objective

Build diplomacy system. Player negotiates with other nations, forms alliances, isolates enemies.

By the end of Wave 8:
- Other nations exist with diplomats
- Player can form alliances
- Treaties work
- Trade agreements benefit both sides
- Espionage possible
- Reputation matters globally
- Ready for Wave 9 (multiplayer)
