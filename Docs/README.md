## Getting Started

**New to the project?**
1. Read [The Republic Manifesto](./Gameplay%20Philosophy/MANIFESTO.md) (2 min)
2. Review [Design Principles](./Gameplay%20Philosophy/Design%20Principles.md) (10 min)
3. Check [Features](./Features/README.md) for what we're building
4. See [Roadmap](./Roadmap/README.md) for timeline

**Making a design decision?**
1. Check the [Republic Test](./Gameplay%20Philosophy/Design%20Principles.md#the-republic-test) in Design Principles
2. Consult relevant [Feature](./Features/README.md) documentation
3. Refer to [UX](./UX/README.md) for interface patterns

**Building a system?**
1. Read [Design Principles](./Gameplay%20Philosophy/Design%20Principles.md) for philosophy
2. Check [Architecture](./Architecture/README.md) for technical patterns
3. Review [TDD](./TDD/README.md) for technical specs
4. Implement per [Features](./Features/README.md) spec
