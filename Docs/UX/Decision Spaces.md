# Decision Spaces

The distinct interfaces where players make meaningful choices.

## Economic Decisions

### Production Management
- Allocate population to industries
- Set production targets
- Manage supply chains

### Trade System
- Negotiate treaties
- Set trade prices
- Manage export/import
- Form trading blocs

### Taxation
- Set tax rates
- Define tax policy
- Allocate revenue
- Manage treasury

## Political Decisions

### Legislative System
- Propose laws
- Vote on legislation
- Manage factions
- Balance interests

### Government Structure
- Appoint officials
- Define positions
- Delegate authority
- Manage succession

### Diplomatic Relations
- Form alliances
- Negotiate treaties
- Manage embassies
- Issue ultimatums

## Military Decisions

### Force Composition
- Build units
- Assign commanders
- Set formations
- Manage equipment

### Strategy & Tactics
- Plan campaigns
- Issue movement orders
- Set engagement rules
- Manage supply lines

### Warfare
- Declare war
- Manage battles
- Negotiate surrender
- Handle peace treaties

## Population Decisions

### Demographics
- Influence population distribution
- Affect migration
- Manage happiness
- Handle unrest

### Cultural Policy
- Set cultural tone
- Influence values
- Manage assimilation
- Handle conflicts

## Information Requirements

Each decision space displays:
- **Current State**: What is the status now?
- **Available Options**: What can the player do?
- **Consequences**: What will happen?
- **Historical Context**: What happened before?
- **Time Impact**: How long will this take?

---

*Each decision space should feel empowering and consequential.*
