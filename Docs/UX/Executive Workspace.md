# Executive Workspace

The primary player interface for managing your nation.

## Overview

The Executive Workspace is the main hub where players:
- View their nation's status
- Access all systems
- Make critical decisions
- Monitor world events
- Plan strategy

## Layout Structure

### Header
- Nation name and flag
- Current date/time
- Pause/play controls
- Time scale selector
- Menu and settings

### Main Content Area
- Map view (when needed)
- Current decision/interface
- Information display

### Sidebar
- Quick access to systems
- Active alerts
- Key metrics

### Bottom Bar
- Time controls
- Save/load
- System status

## Key Metrics Display

### Economic Indicators
- Treasury balance
- GDP trend
- Trade balance
- Inflation rate

### Political Indicators
- Government type
- Stability rating
- Public opinion
- Legislative status

### Military Indicators
- Force strength
- Unit count
- Readiness
- Deployment status

### Population Indicators
- Total population
- Growth rate
- Happiness/satisfaction
- Migration flows

## System Access

From the Executive Workspace, players can quickly access:
- **Economy** - Production, trade, taxation
- **Government** - Laws, elections, succession
- **Military** - Units, strategy, warfare
- **Diplomacy** - Treaties, relations, alliances
- **Population** - Demographics, migration, happiness
- **Reports** - Analytics, history, forecasting

## Interaction Flow

1. Player opens Executive Workspace
2. Views nation status and alerts
3. Selects a system to interact with
4. Makes decisions within that system
5. Returns to workspace
6. Advances time
7. Receives feedback on decisions
8. Repeats

---

*The Executive Workspace is the player's command center.*
