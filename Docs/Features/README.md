# Features\n\nIndividual feature specifications organized by **Production Wave**.\n\n**9 Waves. Each builds on the previous. Each adds new gameplay.**\n\nSee [PRODUCTION_WAVES.md](../../.github/PRODUCTION_WAVES.md) for the complete wave strategy and dependency chain.\n\n## Organization\n\nFeatures are organized by:\n1. **Wave** (Wave 0-9)\n2. **System** (Foundation, Executive Workspace, Simulation, etc)\n3. **Feature** (specific systems within each)\n\n---\n\n## Wave 0: Foundation\n\n**Objective**: Build nothing visible. Only architecture.\n\n**Duration**: 2-3 weeks\n\n**What Player Sees**: Nothing\n\n### Core Systems\n- Project Bootstrap & Build Pipeline\n- Folder Structure & Organization\n- Logging System\n- Configuration Manager\n- Save System (framework)\n- Event Bus (decoupled communication)\n- Time Engine (60 tick/second)\n- Simulation Core (deterministic stepping)\n- Dependency Injection Container\n- Game Loop (update → render → repeat)\n\n### Milestone Gate\n**\"The skeleton works\"** - Project builds, framework ready\n\n---\n\n## Wave 1: Executive Workspace\n\n**Objective**: Build the player's office environment.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Your office for the first time\n\n### Core Systems\n- Campaign HQ (main environment, first room)\n- Office Interaction (movement, object interaction)\n- Phone System (make/receive calls)\n- Email System (read/write emails)\n- Visitor System (NPCs arrive, meetings)\n- Desk (where you sit and work)\n- Calendar (schedule and planning)\n- Notification System (alerts and updates)\n- Transition Manager (moving between locations)\n- Immersive Environment (lighting, sound, atmosphere)\n\n### Player Experience\n- You sit down at your desk\n- Phone rings\n- You take the call\n- Someone visits your office\n- You receive an email notification\n- You check your calendar\n\n### Milestone Gate\n**\"The player has an office\"** - Player immersed in their workspace\n\n**Still no politics. Just the office.**\n\n---\n\n## Wave 2: World Simulation\n\n**Objective**: Build the world that lives without the player.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Reports on their desk about a living world\n\n### Core Systems\n- World Manager (entity lifecycle)\n- Country Entity (nation data model, full attributes)\n- Geography System (terrain, regions, resources, climate)\n- Population System (demographics, population growth, migration)\n- Resources System (supply, demand, distribution)\n- Currency System (exchange rates, value fluctuation)\n- Government Types System (monarchy, democracy, dictatorship, etc)\n- Constitution System (rules that constrain power)\n- Political Culture System (values, traditions, cultural beliefs)\n- Economic Foundation (production chains, basic trade)\n\n### Player Experience\n- \"Your country's GDP is $X\"\n- \"Population grew by 2%\"\n- \"Resource prices are up\"\n- \"Trade is flourishing\"\n- All happening without your input\n\n### Milestone Gate\n**\"The world lives without a player\"** - Autonomous simulation\n\n**You can see the world. You can't change it yet.**\n\n---\n\n## Wave 3: Campaign\n\n**Objective**: Build the electoral system - how you gain power.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Campaign interface, opponents, voters, elections\n\n### Core Systems\n- Campaign Framework (running for office)\n- Electoral System (voting, campaigns, polling)\n- Political Parties (join/leave/create parties)\n- Faction System (political factions with interests)\n- Campaign Mechanics (rallies, advertisements, debates, messaging)\n- Public Opinion (approval ratings, favorability, support levels)\n- Electoral Rules (how elections work in each country)\n- Winning/Losing Conditions (how you win/lose office)\n- Candidate System (rival candidates, their strategies)\n- Transition System (from civilian to office holder)\n\n### Player Experience\n- You announce candidacy\n- You campaign across the country\n- You debate opponents\n- You make campaign promises\n- Election day arrives\n- You win (or lose and restart)\n- If you win, you transition to government\n\n### Milestone Gate\n**\"You can run for office\"** - Electoral system functional\n\n**You're fighting for power. Not yet wielding it.**\n\n---\n\n## Wave 4: Government\n\n**Objective**: Build how you rule after winning office.\n\n**Duration**: 6-8 weeks\n\n**What Player Sees**: Cabinet, ministers, executive decisions\n\n### Core Systems\n- Cabinet System (ministers, advisors, specialists)\n- Executive Powers (what you can decide as leader)\n- Government Interface (how you access each function)\n- Advisory System (ministers propose ideas and policies)\n- Decision System (choose between competing options)\n- Approval System (public opinion tracks your rule)\n- Crisis System (crises emerge from simulation)\n- Legitimacy System (power requires mandate from people)\n- Minister Specialization (finance minister, defense minister, etc)\n- Resource Allocation (distribute budget across needs)\n\n### Player Experience\n- Finance Minister proposes tax policy\n- Defense Minister wants military spending\n- Social Minister wants healthcare funding\n- You have limited resources\n- You choose what to fund\n- Public opinion responds\n- Approval goes up or down\n\n### Milestone Gate\n**\"You have the power to govern\"** - Governing system functional\n\n**Now you're governing. Making difficult choices.**\n\n---\n\n## Wave 5: Economy\n\n**Objective**: Build economic systems and trade.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Production, trade, jobs, markets responding to policy\n\n### Core Systems\n- Production System (how goods are manufactured)\n- Trade System (importing/exporting goods)\n- Market System (prices, supply/demand dynamics)\n- Labor System (employment, wages, job creation/loss)\n- Taxation System (tax rates, revenue collection)\n- Business System (companies, industries, sectors)\n- Income Distribution (wealth gap, inequality)\n- Economic Advisors (economists giving recommendations)\n- Economic Crises System (recessions, booms, crashes)\n- Trade Routes (international commerce paths)\n\n### Player Experience\n- You lower taxes\n- Businesses hire more workers\n- Prices fall\n- People are happier\n- Government revenue drops\n- You have less money for programs\n- You raise taxes\n- Prices spike\n- Unemployment rises\n- People get angry\n\n### Milestone Gate\n**\"Your economy responds to your rule\"** - Economic systems functional\n\n**Trade-offs. Consequences. Economics is complex.**\n\n---\n\n## Wave 6: Legislature\n\n**Objective**: Build parliament, laws, and political competition.\n\n**Duration**: 6-8 weeks\n\n**What Player Sees**: Parliament chamber, bills, voting, opposition\n\n### Core Systems\n- Parliament System (legislature with elected members)\n- Bills System (proposed laws, bill lifecycle)\n- Voting System (how laws pass/fail)\n- Faction System (political parties in parliament)\n- Opposition System (political rivals, competitors)\n- Party Discipline (controlling your party members)\n- Coalition System (building majorities for legislation)\n- Constitutional Constraints (what you CAN'T do)\n- Legislative Process (debate, amendments, voting)\n- Law Effects System (how laws change society)\n\n### Player Experience\n- You propose a law\n- Your party mostly supports it\n- Opposition fights it\n- You negotiate with swing voters\n- Vote is close\n- Law passes by 2 votes\n- You've changed society, but barely\n\n### Milestone Gate\n**\"You must work with parliament\"** - Legislature system functional\n\n**Democracy is messy, slow, but real.**\n\n---\n\n## Wave 7: Military\n\n**Objective**: Build military systems and warfare.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Armed forces, deployments, warfare, casualties\n\n### Core Systems\n- Military Framework (armies, navies, air forces)\n- Military Strategy System (where to deploy forces)\n- Warfare System (how conflicts play out)\n- Defense System (protecting borders)\n- Military Spending (budget for armed forces)\n- Geopolitical Tension (when conflicts emerge)\n- Victory/Defeat in Warfare (winning or losing conflicts)\n- War's Impact (refugees, economy, casualties)\n- Military Intelligence System (spying, reconnaissance)\n- Arms Race System (military buildup dynamics)\n\n### Player Experience\n- Border tensions rising\n- Your general requests authorization\n- You authorize military movement\n- Other side backs down (or doesn't)\n- War breaks out\n- Your side is winning\n- You negotiate peace\n- Victory gives you legitimacy\n\n### Milestone Gate\n**\"You control military power\"** - Military systems functional\n\n**War is the ultimate consequence.**\n\n---\n\n## Wave 8: Diplomacy\n\n**Objective**: Build international relations and treaties.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Other nations, relationships, negotiations\n\n### Core Systems\n- Diplomatic Relations (how other nations view you)\n- Treaty System (signing alliances, agreements)\n- Trade Agreements (formal commerce treaties)\n- Espionage System (spying on other nations)\n- Alliance System (mutual defense pacts)\n- Diplomacy Interface (negotiating with other leaders)\n- International Incidents (disputes between nations)\n- Soft Power System (cultural influence, persuasion)\n- Diplomatic Crises (international conflicts)\n- International Organizations (UN-like bodies)\n\n### Player Experience\n- Neighboring nation threatens war\n- You quickly form alliance with another power\n- Threat backs down\n- You win diplomatically\n- Later, you need trade partners\n- You negotiate lower tariffs\n- Trade increases, economy booms\n\n### Milestone Gate\n**\"You can shape international politics\"** - Diplomacy systems functional\n\n**Diplomacy multiplies your power.**\n\n---\n\n## Wave 9: Multiplayer\n\n**Objective**: Build multiplayer and human player interaction.\n\n**Duration**: 4-6 weeks\n\n**What Player Sees**: Other human players as nation leaders\n\n### Core Systems\n- Networking Protocol (deterministic, turn-based)\n- Server Architecture (persistent world)\n- State Synchronization (all players see same world)\n- Player Sessions (join/leave/resume)\n- Turn System (coordinated turns across players)\n- Emergent Diplomacy (player-to-player negotiation)\n- Player Alliances (humans choosing sides)\n- Player Warfare (humans fighting each other)\n- Betrayal System (breaking agreements)\n- Emergent Storytelling (player-created narratives)\n\n### Player Experience\n- You're allied with Player 2\n- You share intelligence\n- Player 3 attacks you\n- You call Player 2 for help\n- Player 2 joins the war\n- You defeat Player 3\n- Later, Player 2 secretly builds military\n- You realize they're planning to betray you\n- Political intrigue becomes real\n\n### Milestone Gate\n**\"Politics becomes personal\"** - Multiplayer systems functional\n\n**With humans, the game becomes about reputation and trust.**\n\n---\n\n## The Republic Test\n\nBefore implementing any feature, ensure it answers **yes** to most of these:\n\n1. ✅ Does it strengthen the executive leadership experience?\n2. ✅ Does it create meaningful strategic choices?\n3. ✅ Does it affect multiple systems?\n4. ✅ Does it fit within the office-centric design philosophy?\n5. ✅ Does it increase replayability?\n6. ✅ Does it create opportunities for emergent storytelling?\n7. ✅ Does it reinforce the player's role as Head of Government?\n\nIf it fails most of these questions, **redesign or omit it**.\n\n---\n\n## Dependency Chain\n\n**Why this order matters:**\n\n```\nYou can't have Cabinet Meetings\n  without Government\n\nYou can't have Government\n  without Campaign\n\nYou can't have Campaign\n  without Countries\n\nYou can't have Countries\n  without Simulation\n\nYou can't have Simulation\n  without Time\n```\n\nEach Wave **enables** the next Wave.\n\nNo shortcuts. No skipping.
